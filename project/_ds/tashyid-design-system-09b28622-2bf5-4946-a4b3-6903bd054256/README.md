# Tashyid Design System

## Overview

**Tashyid** (تشييد) is a Saudi Arabian real estate and construction company. The name means "construction" or "building" in Arabic, reflecting the company's core identity of building lasting legacies through land investment and development.

**Slogan (2026):** "FROM LAND TO LEGACY" / "أرض تُستثمر.. إرث يَدوم"  
*(A land that's invested… a legacy that endures)*

The brand is bilingual (Arabic + English), rooted in a premium, authoritative visual identity that blends Islamic/Arabic design heritage with clean modernism.

## CONTENT FUNDAMENTALS

### Voice & Tone
- **Premium and authoritative** — language reflects the gravitas of legacy building, land, and long-term investment
- **Bilingual by default** — Arabic is primary, English supports; they appear together on most communications
- **Formal in Arabic, clean and bold in English** — Arabic copy has poetic/rhetorical qualities (e.g., "إرث يَدوم" = "a legacy that endures"); English is bold, declarative, and uppercase for headlines
- **No emoji** — the brand does not use emoji in any visual communications
- **"You" orientation** — addresses the audience with aspiration ("your land", "your legacy") rather than "I"
- **Casing in English:** Headlines are ALL CAPS or Title Case; body copy uses sentence case
- **Brevity is key** — headlines are short and impactful; no verbose copy

### Examples
- Headline EN: "FROM LAND TO LEGACY"
- Headline AR: "أرض تُستثمر.. إرث يَدوم"
- Tagline style: Paired bilingual — English leads, Arabic echoes

## VISUAL FOUNDATIONS

### Color System
| Role | Name | Hex | Notes |
|---|---|---|---|
| Primary | Burgundy | `#7A1143` | Pantone P77-16U; used for logo, key UI elements, type |
| Secondary | Light Green | `#AAD3BF` | Pantone P136-10C; supporting accent, nature motifs |
| Secondary | Near-Black | `#2E2D2C` | Pantone P179-15C; body text, dark backgrounds |
| Neutral | White | `#FFFFFF` | Primary background; reversed logo context |

All three brand colors are used at 10–90% opacity for tints and overlays.

### Typography
- **Arabic font:** DIN Next Arabic (Regular 400, Medium 500, Bold 700) — OTF files in `fonts/`
- **English font:** DIN Next LT W23 Regular — OTF file in `fonts/` (Regular weight only; bold/medium synthesized by browser)
- Both fonts are from the DIN Next family — geometric, clean, slightly industrial
- Arabic text is set right-to-left (RTL); documents/layouts adapt accordingly
- Body text is fully justified in Arabic (matching brand guide samples)

### Logo
The Tashyid logo is a geometric Arabic wordmark (تشييد) whose letterforms are abstracted into an architectural roofline silhouette — diagonal strokes create a mountain/peaks motif.

**Variants:**
- `TD-LOGO-02-badge-arabic.png` — Burgundy square badge + Arabic wordmark below (primary)
- `TD-LOGO-03-badge-english.png` — Burgundy square badge + "TASHYID" Latin wordmark below
- `TD-LOGO-04-badge-white.png` — White badge version (for dark backgrounds)
- `TD-LOGO-05-wordmark-white.png` — Horizontal white wordmark only
- `TD-LOGO-06-wordmark-burgundy.png` — Horizontal burgundy Arabic-only wordmark

**Clearspace:** maintain minimum clearspace equal to the height of the Arabic letter "ت" on all sides.

### Patterns
Patterns are used as background textures, decorative dividers, and surface fills:

| File | Description | Usage |
|---|---|---|
| `TD-PATTERNS-01` | White/transparent (light shield repeat) | Subtle background on white surfaces |
| `TD-PATTERNS-02` | Palm tree + diamond repeat, burgundy outline | Decorative backgrounds, collateral |
| `TD-PATTERNS-03` | White version of palm tree pattern | For use on colored surfaces |
| `TD-PATTERNS-04` | Shield/logo icon repeat, white on white | Ultra-subtle texture on white |
| `TD-PATTERNS-05` | Zigzag chevron gradient, burgundy ombre | Dividers, footers, section breaks |

### Backgrounds
- **Primary backgrounds:** White (`#FFFFFF`), Off-white (`#F8F6F4`)
- **Dark/hero backgrounds:** Burgundy (`#7A1143`) with white logo/text
- **Textured backgrounds:** Pattern overlays at 10–30% opacity over white or burgundy
- **No photographic backgrounds** evident in brand assets; patterns are preferred
- **No gradients** on solid color fields — chevron pattern (`TD-PATTERNS-05`) provides the gradient feel

### Spacing & Layout
- Clean, generous whitespace — premium brands breathe
- Content is centered or RTL-aligned
- Diamond accent shape (◇) used as a decorative/pagination element
- Grid: 12-column; generous margins (min 48px on mobile, 80px+ on desktop)

### Borders & Radii
- Very minimal use of borders — thin 1px lines in burgundy for dividers
- Corner radii: very subtle (2–4px) or none; the brand leans geometric, not rounded
- Cards: minimal shadow, thin border or none, square-to-slightly-rounded corners

### Shadows
- Subtle: `0 4px 16px rgba(46,45,44,0.10)` — used sparingly on elevated cards
- The brand is flat-forward; heavy shadows are inconsistent with the aesthetic

### Hover & Interaction States
- Hover: slightly lighter burgundy (`#9e2d62`) or opacity reduction to 80%
- Press/active: darker burgundy (`#5a0c32`)
- Transitions: smooth ease, ~250ms — no bouncy/springy animations
- No aggressive motion; brand tone is measured and confident

### Animation
- Minimal and purposeful — fades, gentle slides
- Duration: 200–400ms, ease-out preferred
- No playful or bouncy easings; no looping decorative animations

### Iconography
- No dedicated icon set was provided or identified in brand assets
- Diamond shape (◇) is the brand's decorative accent/icon — used as bullet, page number surround, divider
- Palm tree motif appears in patterns — not as a standalone icon
- No emoji usage
- For UI icons, recommend Phosphor or Feather icons (thin/regular weight) in burgundy or near-black

### Imagery Guidance
- No photography provided in brand assets
- Implied aesthetic: warm-toned, architectural photography; aerial or ground-level land/property shots
- Color treatment: warm, slightly desaturated; no heavy filters
- No grain treatment evident from provided assets

## ICONOGRAPHY

No proprietary icon set was provided. The Tashyid brand uses:

1. **Diamond accent (◇)** — This is the brand's signature decorative element, appearing as page numbers, bullet points, and section dividers. It is drawn as a small outlined diamond.
2. **Palm tree motif** — Appears exclusively in repeat patterns (`TD-PATTERNS-02`, `TD-PATTERNS-03`). Not used as a standalone icon.
3. **Chevron/zigzag** — The roofline/mountain silhouette from the logo is echoed in the chevron pattern (`TD-PATTERNS-05`).

**Recommended CDN icon set:** [Phosphor Icons](https://phosphoricons.com/) — Regular/Light weight, in `#7A1143` or `#2E2D2C`.

No icon font files or SVG sprites were provided.
